Search syntax

Unix-style:

  • * matches everything,
  • [seq] matches any character in seq
  • [!seq] matches any character not in seq
    e.g. type *Weinberg* to search for anything containing "Weinberg"

Regulars expressions:

https://docs.python.org/3/library/re.html\#module-re 

    For example:
  • Isaac (?!Asimov) will match "Isaac " only if it is not followed by "Asimov".
  • weinb.*cosm will search for anything containing "weinb" followed by any sequence of characters further folowed by the sequence "cosm"

Tags Notation:

  • The names starting with G: are names of Bibdesk static groups. They can be created by issuing ./bibgroups2keywords.py -b myfile.bib -k -a
